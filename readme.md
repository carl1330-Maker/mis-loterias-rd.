# 🎰 Loterías RD - Resultados en Vivo

![Loterías RD](https://img.shields.io/badge/status-activo-brightgreen)
![GitHub last commit](https://img.shields.io/github/last-commit/tu-usuario/tu-repositorio)
![GitHub pages](https://img.shields.io/badge/GitHub-Pages-blue)

Página web interactiva que muestra los resultados simulados de las principales loterías de República Dominicana. Los números se actualizan automáticamente y mantienen un historial de resultados anteriores.

## ✨ Características

- 🎯 *8 Loterías Dominicanas*: Loteka, Lotería Real, Nacional, La Primera, Suerte, Tropical, Leidsa y King
- 🔄 *Actualización Automática*: Cada 30 segundos se generan nuevos números
- 📊 *Historial Persistente*: Guarda los últimos 5 resultados de cada lotería
- 💾 *Almacenamiento Local*: Los datos se guardan en tu navegador
- 🎨 *Diseño Moderno*: Interfaz fresca con efectos glassmorphism y animaciones
- 📱 *Responsive*: Se ve perfecto en móviles, tablets y desktop
- ⚡ *Rendimiento*: 100% HTML, CSS y JavaScript puro, sin dependencias
